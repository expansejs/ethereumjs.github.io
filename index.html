expansejs

Your Javascript gateway to Expanse

This is a collection of libraries and utilities for Expanse.

Use cases

Creating an online wallet?

Check out keyxpanse or expansejs-wallet (with HD wallet support) for managing keys and expansejs-tx for creating transactions with them. expansejs-icap might also come handy for dealing with the ICAP (Expanse in IBAN) format.

Creating a Dapp?

You will need to interface with the Expanse network. web3.js provides a complete Javascript API to interact with the RPC interface. If looking for a more lightweight option, expansejs-abi or solidity.js can handle the ABI encoding.

Interested in running a node?

See node-blockchain-server. It is in a pretty rough state at the moment, but at least can download the blockchain.

Full list of repos

  • browser-builds: browser builds of expansejs libraries
  • common: the genesis data for the blockchain
  • ethashjs: Ethash in Javascript
  • expansejs-abi: ABI encoding and decoding
  • expansejs-account: account schema encoding, decoding and validation
  • expansejs-block: block schema encoding, decoding and validation
  • expansejs-blockchain: manage a blockchain
  • expansejs-codesim: run EVM or Solidity code and examine the output
  • expansejs-icap: utilities for handling ICAP (Expanse in IBAN) encoding
  • expansejs-lib: meta package for loading the other expansejs- modules
  • expansejs-testing: transforms the official test vectors to a format suitable for expansejs
  • expansejs-tx: transaction creation, manipulation, signing and verification
  • expansejs-units: Expanse unit conversion
  • expansejs-util: a collection of frequently used methods by the other libraries
  • expansejs-wallet: lightweight toolkit for managing Expanse keys, including HD wallet support
  • expansejs-vm: a complete EVM (Ethereum Virtual Machine) and state processing implementation
  • geth.js: start and stop geth from Node.js
  • helpeth: purists' commandline tool for key and transaction management
  • keyxanse: create, import and export Expanse keys
  • merkle-patricia-tree: This is an implementation of the modified merkle patricia tree as specified in the Ethereum yellow paper
  • node-blockchain-server: aims to provide a full Expanse node implementation
  • node-devp2p: implementation of the RLPx transport protocol for Expanse (used between nodes)
  • node-devp2p-dpt: implementation of the RLPx DPT (peer table) protocol for Expanse
  • node-devp2p-eth: implementation of the Expanse sub-protocol over RLPx
  • organization and ideas: plans and discussions
  • rlp: RLP (Recursive Length Prefix) encoding and decoding
  • testrpc: fast Expanse RPC node for testing and development

Expanse JS projects not tracked here

  • web3.js: the complete API as seen in the wiki
  • solidity.js: ABI encoding and decoding (the relevant code split out from web3.js)
